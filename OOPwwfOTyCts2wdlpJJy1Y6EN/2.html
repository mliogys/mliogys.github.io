Class is a blueprint or template or set of instructions to build a specific type of object with specific behavior. Object - instance of a class - a created variable with properties defined in a class. If class is called a blueprint then it's object can be called actual item created as defined in blueprint. Class definition includes:

  • Attributes. Class variables describing properties.
  • Methods. Class functions describing behaviour.
  • Constructors. Class initializers describing properties values setting rules.

All attributes, methods and constructors are called together class members. Let's take a look at class creation syntax.

Object is created by calling the constructor. In C# and Java programming languages constructor's name must match class name, in Python - constructor is named as __init__, but still the class name is used to call it (see example bellow).

Object initialization syntax:
1. ClassName objectName = new ClassName();
Object initialization example:
1. Circle circle1 = new Circle();
2. Circle circle2 = new Circle();
3. Circle circle3 = new Circle();

In last example we have created three objects circle1, circle2 and circle3. These are three different objects (stored in separate places of memory), but the inner side (attribute values) are identical. In last example we have created three circles of radius value equal to 1.0. Usually, in practice, we need to create objects using different set of initial data. In such case, we need to update a class with new constructor accepting value from outside and assigning that value to appropriate attribute.
