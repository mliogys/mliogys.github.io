Class - blueprint or template or set of instructions to build a specific type of object with specific behavior. Object - instance of class - a created variable with properties defined in a class. If class is called a blueprint then it's object can be called actual item created as defined in blueprint. Class definition includes:

  • Attributes. Class variables describing properties.
  • Methods. Class functions describing behaviour.
  • Constructors. Class initializers describing properties values setting rules.

All attributes, methods and constructors are called together class members. Let's take a look at class creation syntax.

Objektas yra sukuriamas, iškviečiant klasėje aprašytą konstruktorių. C# ir Java kalbose konstruktoriaus pavadinimas privalo sutapti su klasės pavadinimu, o Python kalboje konstruktorius yra pavadinimu __init__, tačiau kuriant objektą naudojamas klasės pavadinimas.

Objektų kūrimo pavyzdyje, sukūrėme tris objektus circle1, circle2 ir circle3. Tai yra trys skirtingi objektai (skirtingose atminties vietose saugoma informacija), tačiau jų vidinė informacija (atributų reikšmės) yra vienoda. Pavyzdžio atveju, sukūrėme tris apskritimus, kurių spindulys yra 1.0. Realioje praktikoje, retai pasitaikantis atvejis, kuomet reikalinga, kad visi tam tikros klasės ojektai būtų kuriami su ta pačia pradine informacija.
