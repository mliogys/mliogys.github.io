Object-oriented programming - programming paradigm based on objects, that, in a form of attributes, store data and, in a form of functions, store data processing code. This is the key paradigm that is used nowadays in software development. So, it is essential for you as software engineers and developers to understand object-oriented programming principles.

In order to apply object-oriented programming principles in practice, we need to choose the right programming language and development environment. At the moment, there are hundreds of programming languages. Here are some resources, that can help you to decide which programming language to choose:

Stack Overflow GitHub Coding Dojo

In this course, you will find prepared examples in three programming languages: C#, Java and Python. Recommended IDE (Integrated Development Environments):

Visual Studio Community (C#) IntelliJ IDEA (Java) JetBrains PyCharm Community (Python)
