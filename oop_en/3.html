3.1. Data types

All the data processed in software are stored in main memory. The addresses to data are symbolized as variables in source code. Each variable is an instance of some kind (data type). Data type has its own properties and behaviour (operations) predefined. In some cases, same operation can be implemented differently in different types, for example, addition operation on two integers will produce the sum of two integers as a result, and will produce quite different result on two strings.

Every programming language has a set of standard data types, that can be used to design and create composite data types: structures or classes.

3.2. Conditionals

Every software developed to solve real life problem is unimaginable without using any of condition testing operations (conditionals). There are two the most popular conditional statements, in any programming language, and they are: if and switch.

~Conditional statement if

Testable condition can be one of two: to be true or to be false. If tested condition's result is true, then all operations listed in then branch are executed , if false - else branch.

Syntax:
1.if (condition)
2.{
3.// condition test result is true;
4.}
5.else
6.{
7.// condition test result is false;
8.}
Example:
1.if (x < y)
2.{
3.WriteLine($"Number {x} is less than {y}");
4.}
5.else
6.{
7.WriteLine($"Number {x} is not less than {y}");
8.}

Quite common practice to test several conditions at the same time using the same sentence. In such case, we need to combine testable conditions to one composite statement. There are two ways to combine conditions:

  • Conjunction, readable as „and”. Conjunction result is true, then all testable conditions gives result true. In C#, Java conjunction symbol is a pair of ampersand symbols, i.e. &&, in Python - keyword and.
  • Disjunction, readable as „or”. Disjunction result is true, then at least one of testable conditions gives result true. In C#, Java disjunction symbol is a pair of vertical bars, i.e. ||, in Python - keyword or.

Conjunction example:
1. if (x > y && x > z)
2. {
3. WriteLine($"{x} is larger than {y} and {z}");
4. }
Disjunction example:
1. if (x > y || x > z)
2. {
3. WriteLine($"{x} is not the smallest number");
4. }

To test if object is initialized (storing actual data), in C# and Java we need to use if (objectName == null), in Python - if objectName is None:.

To test if object is an instance of some specific class, in C# we need to use if (objectName is ClassName), in Java - if (objectName instanceof ClassName), in Python - if objectName is ClassName:.

Conditional statement switch

Statement switch is convenient to use if there are limited number of alternatives. It can be used in menu based systems, checking enum values and etc. You can use switch statement in C# and Java, unfortunately, no such operator exists in Python.

2. {
3. case value1:
4. // set of statements1;
5. break;
6. case value2:
7. // set of statements2;
8. break;
9. default:
10.// set of statements for any other value;
11.break;
12.}

3.3. Loops

Solving real life problems, usualy, we need to implement in source code repetetive operations. In order, to avoid repetetive operations, we can use, or allow me to say must, use looping statements. Looping statement - operator, automatically repeating operations till predefined condition is valid. One full round of operations in looping statement is called an iteration. Traditionally, we can find for and while looping statements in every programming language. The principles are the same, only the syntax may differ in different languages.

for syntax
1.for (expression1; expression2; expression3)
2.{
3.// expression1 - loop variable initialization
4.// expression2 - looping condition
5.// expression3 - loop variable value change rule
6.}
for example:
1.for (int i = 1; i <= 10; i++)
2.{
3.WriteLine($"i = {i}");
4.}
while syntax:
1.while (condition)
2.{
3.// statements;
4.// condition variable's value change rule;
5.}
while example:
1.int i = 1;
2.while (i < 11)
3.{
4.WriteLine($"i = {i}");
5.i++;;
6.}
