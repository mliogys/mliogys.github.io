3.1. Duomenų tipai

Programos apdorojami duomenys yra saugomi darbinėje atmintyje. Duomenų adresą programos kode simbolizuoja kintamieji. Kiekvienas kintamasis yra tam tikros klasės objektas. Operacijos, galimos atlikti su kintamaisiais, yra apibrėžtos klasėje pagal kurią kintamasis yra sukurtas. Ta pati operacija skirtingiems duomenų tipams gali duoti skirtingą rezultatą. Pavyzdžiui, sudėties operacija, skaičių atveju, grąžiną sumą, o tekstinės informacijos atveju, tekstas apjungiamas.

Kiekviena programavimo kalba turi standartinių duomenų tipų rinkinį, kurių pagrindu gali būti konstruojami kiti sudėtiniai duomenų tipai: struktūros arba klasės.

3.2. Sąlygos operatoriai

Programose, skirtose spręsti realius gyvenimiškus uždavinius neišsiverčiama be įvairių sąlygų tikrinimo ir vykdomų operacijų, esant vienai ar kitai situacijai, aprašymo. Sąlygos operatoriai palengvina šią užduotį supaprastindami užduočių valdymą. Populiariausiose programavimo kalbose dažniausiai sutinkami du situacijų valdymo operatoriai: if ir switch.

Operatorius if

Šio operatoriaus principas yra vienodas visose programavimo kalbose - aprašytai sąlygai yra dvi alternatyvos: tiesa arba netiesa. Jei rezultatas yra tiesa, tuomet vykdomos then šakos operacijos, jei netiesa - else šakos operacijos.

Sintaksė:
1.if (sąlyga)
2.{
3.// sąlygos reikšmė tiesa;
4.}
5.else
6.{
7.// sąlygos reikšmė netiesa;
8.}
Pavyzdys:
1.if (x < y)
2.{
3.WriteLine($"Skaičius {x} yra mažesnis už {y}");
4.}
5.else
6.{
7.WriteLine($"Skaičius {x} yra nemažesnis už {y}");
8.}
Sintaksė:
1.if (sąlyga) {
2.// sąlygos reikšmė tiesa;
3.} else {
4.// sąlygos reikšmė netiesa;
5.}
Pavyzdys:
1.if (x < y) {
2.format("Skaičius %d yra mažesnis už %d", x, y);
3.} else {
4.format("Skaičius %d yra nemažesnis už %d", x, y);
5.}
Sintaksė:

2.# sąlygos reikšmė tiesa
3.else:
4.# sąlygos reikšmė netiesa
Pavyzdys:

2.print("Skaičius",x,"yra mažesnis už",y)

4.print("Skaičius",x,"yra nemažesnis už",y)

Labai dažnai susiduriama su situacija, kai reikia tikrinti ne vieną, o kelias sąlygas iš karto. Tokiu atveju, tikrinamas sąlygas reikia apjungti. Sąlygų jungimo būdai yra šie:

  • Konjunkcija (loginė sandauga), skaitoma „ir”. Konjunkcijos bendras rezultatas yra tiesa, kai visų sąlygų rezultatas yra tiesa. C#, Java kalbose konjunkcija žymima && simbolių pora, Python kalboje - raktiniu žodžiu and.
  • Disjunkcija (loginė suma), skaitoma „arba”. Disjunkcijos bendras rezultatas yra tiesa, kai bent vienos sąlygos rezultatas yra tiesa. C#, Java kalbose disjunkcija žymima || simbolių pora, Python kalboje - raktiniu žodžiu or.

Konjunkcijos pavyzdys:
1. if (x > y && x > z)
2. {
3. WriteLine($"{x} yra didesnis už {y} ir {z}");
4. }
Disjunkcijos pavyzdys:
1. if (x > y || x > z)
2. {
3. WriteLine($"{x} nėra mažiausias skaičius");
4. }
Konjunkcijos pavyzdys:
1. if (x > y && x > z) {
2. println("%d yra didesnis už %d ir %d", x, y, z);
3. }
Disjunkcijos pavyzdys:
1. if (x > y && x > z) {
2. println("%d nėra mažiausias skaičius", x);
3. }
Konjunkcijos pavyzdys:
1.if x > y and x > z:
2.print("Skaičius",x,"yra didesnis už",y,"ir",z)
Disjunkcijos pavyzdys:
1.if x > y or x > z:
2.print("Skaičius",x,"nėra mažiausias skaičius")

Norint patikrinti ar kintamasis yra objektas „saugantis” tam tikrus duomenis, C# ir Java kalbose naudojama sąlyga if (objektas == null), o Python kalboje - if objektas is None:.

Norint patikrinti ar kintamasis yra nuoroda į tam tikros klasės duomenis, C# kalboje naudojama sąlyga if (objektas is Klasė), Java kalboje naudojama sąlyga if (objektas instanceof Klasė), o Python kalboje - if objektas is Klasė:.

Operatorius switch

Operatorių switch patogu naudoti, kai galimų alternatyvų yra baigtinis kiekis. Alternatyva gali būti pasirinkimas meniu sistemose, enum reikšmė ir pan. Operatorius switch yra prieinamas C# ir Java kalbose, o Python kalboje analogiško operatoriaus nėra.

Sintaksė:
1. switch (kintamasis)
2. {
3. case reikšmė1:
4. // operacijos1;
5. break;
6. case reikšmė2:
7. // operacijos2;
8. break;
9. default:
10.// operacijos, esant kitoms reikšmėms;
11.break;
12.}

3.3. Ciklai

Sprendžiant realius uždavinius, dažnai reikia atlikti daug tų pačių operacijų su skirtingais duomenimis. Tam, kad nekartoti daug kartų programos kode tų pačių operacijų naudojami ciklo operatoriai. Ciklo operatorius - operatorius, kartojantis tam tikrą kiekį kartų operacijas, kol tenkinama iš anksto apibrėžta sąlyga. Vienas kartas atliktų ciklo operacijų vadinamas iteracija. Tradiciškai, beveik kiekvienoje programavimo kalboje galime rasti for ir while operatorius. Veikimo principas vienodas visose kalbose, kai kuriais atvejais, skiriasi tik užrašymo sintaksė.

for sintaksė:
1.for (išraiška1; išraiška2; išraiška3)
2.{
3.// išraiška1 - ciklo kintamojo inicializavimas
4.// išraiška2 - ciklo kartojimo sąlyga
5.// išraiška3 - ciklo kintamojo reikšmės kitimo dėsnis
6.}
for pavyzdys:
1.for (int i = 1; i <= 10; i++)
2.{
3.WriteLine($"i = {i}");
4.}
while sintaksė:
1.while (sąlyga)
2.{
3.// operacijos;
4.// sąlygos kintamojo reikšmės kitimo dėsnis;
5.}
while pavyzdys:
1.int i = 1;
2.while (i < 11)
3.{
4.WriteLine($"i = {i}");
5.i++;;
6.}

C# kalboje yra dar ciklas foreach ir do-while. Ciklą foreach patogu naudoti, kai turimas koks nors duomenų rinkinys (masyvas, sąrašas ir pan.) su kurio elementais reikia atlikti tam tikras operacijas. Ciklas do-while labai panašus į ciklą while yra tik vienas esminis skirtumas: ciklo while atveju pirmiau tikrinama sąlyga ir tik tada vykdomos operacijos (jei sąlygos reikšmė tiesa), ciklo do atveju atvirkščiai, pirmiau įvykdomos iteracijos operacijos, o tik po to tikrinama sąlyga.

4.}
